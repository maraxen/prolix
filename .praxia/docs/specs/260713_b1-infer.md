# B1-INFER

## Goal
Make Claim-1 / B1-full **measurable** by adding an inference step axis that does not stack `(n_steps, N, 3)`, and wire existing XR-DEDUP / XR-SINK-XTC helpers into `EnsemblePlan` for that path.

## Locked decisions

| Topic | Lock |
|-------|------|
| Default EnsemblePlan | Unchanged: `run_mode="trajectory"` → `lax.scan` + full traj (pathological baseline) |
| Inference | `run_mode="inference"` → `dispatch_n_steps_inference` (`lax.while_loop`, carry-only); **not** reverse-mode AD safe |
| Return contract | Inference `Trajectory.positions` shape `(1, n_atoms, 3)` (final frame) |
| Multi-bundle | Host `partition_bundles_by_shape` → K classes; within class Vmap/SafeMap with distinct seeds. DedupGather **not** used for seeded MD (see B1-XTRAX-WIRE). |
| XTC | Inference streams via `io_callback` → `XtcFrameSink` every `save_every`; trajectory mode keeps post-run flush |
| B1 harness | `--path {baseline,inference}`; default **inference**; bathos tag `path:inference`; slurm env `B1_PATH` (never `PATH`) |

## Acceptance Criteria
1. while_loop n_steps final position agrees with scan final (atol) on tiny carry / bundle.
2. Four identical synthetic bundles → `build_dedup_spec_by_shape` has `k=1` and `dispatch_n_mols_dedup` body runs once.
3. Inference + `save_every` writes XTC readable by proxide.
4. `b1_init_exec.py --smoke --path inference` exits 0 with finite metrics and `path=inference` in JSON.
