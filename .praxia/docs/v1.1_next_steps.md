### ~~2. NPT KE Initialization Bug~~ — **RESOLVED (2026-06-01, commit b6e5bb9)**

**Fix**: One-character bug at `settle.py:1944`. Sprint 14 introduced `momentum = momentum / mu`
(inverse scaling) when Bernetti-Bussi 2020 / Parrinello-Rahman require `p' = mu * p`. The `/mu`
amplified KE by 1/mu² per step during box contraction, producing T ≈ 5000–7000 K at the first
NPT record. Changed to `momentum = momentum * mu`. Secondary frame-mixing fix at line 1985
(`positions_old` → `position`). xfail removed from `test_npt_20ps_liquid_water`.

*Long-trajectory NPT divergence (beyond ~10 ps) is a separate issue — root cause is CSVR+SETTLE
KE feedback, addressed in Phase 6.*

**Why now**: LFMiddle falsification makes this the only viable path to dt ≥ 1.0 fs. The paper
critical path requires Phase 5 to start no later than after P1a MolecularBundle — it sits on
the 6-week critical path (P0 → P1 → P2b → P3 → P5 → paper).

**Deliverables**:
- Redesigned Langevin coupling: per-DOF friction+noise (unconstrained DOF only)
- Validation: NVT temperature stability at dt ∈ {0.5, 1.0, 2.0} fs over 50+ ps

**Estimated effort**: 4–6 days  
**Risk**: Medium — requires integrator-thermostat redesign; if coupling is non-local, scope expands.
